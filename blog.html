1. What is DOM and what is the purpose of DOM?

DOM-Document Object Model.An application programming interface (API) for HTML and XML documents is known as a DOM. It specifies how a document is handled and processed and how it is logically structured.When the browser creates DOM of a web-page,the web page is loaded.
The main purpose of the DOM is to maintain a common, language-independent programming interface for HTML and XML documents. By establishing and maintaining this standard, programmers can use any language or development environment to generate, navigate, and update these contents of documents effectively.

2. How will you select HTML elements using DOM? Name the DOM methods.

I can select HTML elements using five DOM methods: 1. getElementsByTagName()-It search element by tag name(e.g:p,div,li). 2. getElementById()-It search element by element_id. 3. getElementsByClassName()-It search element by class name. 4. querySelector()-It returns the first element that matches the specified selector. 5. querySelectorAll()-It returns elements that match the specified selector.

3. What is event bubble?

Event Bubble is a Document Object Model concept . The event bubbles is propagated to the element's parent and grandparent elements in the DOM tree until it reaches the root element.It is a procedure that begins with the element that caused the event and then cascades up to the elements in the tree that contain it.

4. What is a callback function and why will you use it?

A callback function is to run code in response to an event.A callback function is a function that is supplied as an input to another function, which is subsequently called inside the outer function to finish a task.
